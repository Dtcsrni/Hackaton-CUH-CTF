# Subida de archivos ansiosa

## Propósito didáctico
Practicar hardening de un cargador de archivos típico.

## Dinámica de resolución
El alumno corrige el módulo de subida y valida la configuración localmente.

## Material del alumno
- Bundle final: `subida_de_archivos_ansiosa_bundle.zip`
- Página de apoyo: `/subida-de-archivos-ansiosa`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{subida_de_archivos_endurecida}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
El reto se resuelve por análisis y parche; no se sube ningún archivo real al servidor del evento.
