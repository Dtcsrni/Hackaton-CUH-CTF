# Consulta concatenada

## Propósito didáctico
Reforzar el patrón correcto de prepared statements en autenticación.

## Dinámica de resolución
El alumno inspecciona el archivo PHP, corrige la consulta y ejecuta el validador local del bundle.

## Material del alumno
- Bundle final: `consulta_concatenada_bundle.zip`
- Página de apoyo: `/consulta-concatenada`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{consulta_concatenada_corregida}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
No requiere servicio remoto; toda la práctica ocurre en el material descargable y en la validación local.
