# XOR de respaldo

## Propósito didáctico
Introducir reutilización de keystream y lectura de patrones en cifrados relacionados.

## Dinámica de resolución
El alumno compara dos salidas, usa la pista del encabezado y completa un archivo de respuestas.

## Material del alumno
- Bundle final: `xor_de_respaldo_bundle.zip`
- Página de apoyo: `/xor-de-respaldo`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{xor_reutilizado_identificado}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto de análisis conceptual con artefactos sintéticos; no requiere infraestructura externa.
