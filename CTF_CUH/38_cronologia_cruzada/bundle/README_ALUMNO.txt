Cronología cruzada
==========================

Objetivo
--------
Reconstruye la cronología, identifica el actor pivote y el artefacto correcto.

Qué contiene el bundle
----------------------
Publicaciones, agenda, comentarios y validador.

Qué debes entregar
------------------
Completa `respuesta.txt` y ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
