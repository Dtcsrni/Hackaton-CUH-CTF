# Licencia bajo revisión

## Propósito didáctico
Escalar el reversing hacia validaciones segmentadas y lectura más metódica del ejecutable.

## Dinámica de resolución
El alumno analiza el binario, reconstruye la licencia y valida la respuesta localmente.

## Material del alumno
- Bundle final: `licencia_bajo_revision_bundle.zip`
- Página de apoyo: `/licencia-bajo-revision`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{licencia_reconstruida_sin_parche}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
El enfoque sigue siendo reconstrucción de lógica, no modificación binaria del sistema.
