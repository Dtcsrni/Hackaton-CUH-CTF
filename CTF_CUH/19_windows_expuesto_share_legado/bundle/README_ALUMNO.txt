Windows expuesto: share legado
==========================

Objetivo
--------
Identifica el grupo demasiado amplio, el permiso dominante y la corrección prioritaria.

Qué contiene el bundle
----------------------
ACLs exportadas, política operativa y validador.

Qué debes entregar
------------------
Completa `respuesta.txt` y ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
