Cookie de rol heredada
==========================

Objetivo
--------
Endurece la cookie de rol para que el backend no acepte integridad ni valores arbitrarios.

Qué contiene el bundle
----------------------
Módulo Python y notas de formato.

Qué debes entregar
------------------
Corrige `app/cookies.py` y ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
