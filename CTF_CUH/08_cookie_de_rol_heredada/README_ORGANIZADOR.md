# Cookie de rol heredada

## Propósito didáctico
Practicar revisión de cookies de autorización y validación segura de integridad.

## Dinámica de resolución
El alumno corrige un módulo Python y valida la lógica localmente.

## Material del alumno
- Bundle final: `cookie_de_rol_heredada_bundle.zip`
- Página de apoyo: `/cookie-de-rol-heredada`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{cookie_de_rol_endurecida}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
El reto no necesita navegador ni backend vivo; solo código y criterio defensivo.
