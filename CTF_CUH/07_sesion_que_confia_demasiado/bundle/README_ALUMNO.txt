Sesión que confía demasiado
==========================

Objetivo
--------
Haz que la sesión derive el rol desde datos internos y no desde la petición del cliente.

Qué contiene el bundle
----------------------
Backend Python y contexto mínimo del modelo de roles.

Qué debes entregar
------------------
Edita `app/session.py` y valida localmente.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
