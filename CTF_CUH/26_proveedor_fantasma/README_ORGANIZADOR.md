# Proveedor fantasma

## Propósito didáctico
Practicar verificación documental y correlación de identidad corporativa en OSINT.

## Dinámica de resolución
El alumno cruza factura, dominio y nota de compras para emitir una respuesta breve y sustentada.

## Material del alumno
- Bundle final: `proveedor_fantasma_bundle.zip`
- Página de apoyo: `/proveedor-fantasma`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{proveedor_fantasma_correlacionado}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Todo el material es falso y está diseñado para ejercicios de validación de proveedores.
