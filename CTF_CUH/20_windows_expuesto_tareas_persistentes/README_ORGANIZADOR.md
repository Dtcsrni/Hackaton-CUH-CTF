# Windows expuesto: tareas persistentes

## Propósito didáctico
Trabajar persistencia en Windows desde análisis de artefactos y contención.

## Dinámica de resolución
El alumno revisa XML y eventos, completa respuestas y valida localmente.

## Material del alumno
- Bundle final: `windows_expuesto_tareas_persistentes_bundle.zip`
- Página de apoyo: `/windows-expuesto-tareas-persistentes`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{persistencia_en_tareas_reconstruida}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto de forense defensivo sobre tareas programadas.
