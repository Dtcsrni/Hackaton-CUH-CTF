# Semilla predecible

## Propósito didáctico
Practicar selección correcta de fuentes de entropía para llaves y tokens.

## Dinámica de resolución
El alumno sustituye el generador predecible por una fuente segura y valida el resultado.

## Material del alumno
- Bundle final: `semilla_predecible_bundle.zip`
- Página de apoyo: `/semilla-predecible`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{entropia_fuerte_para_llaves}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto de parche centrado en aleatoriedad criptográfica y diseño de APIs simples.
