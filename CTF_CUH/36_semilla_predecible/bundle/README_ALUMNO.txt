Semilla predecible
==========================

Objetivo
--------
Reemplaza la generación predecible por una fuente segura y conserva la salida hexadecimal.

Qué contiene el bundle
----------------------
Módulo de keygen, requisitos de formato y validador.

Qué debes entregar
------------------
Edita `app/keygen.py` y ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
