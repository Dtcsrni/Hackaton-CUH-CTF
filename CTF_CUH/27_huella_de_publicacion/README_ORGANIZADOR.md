# Huella de publicación

## Propósito didáctico
Practicar reconstrucción de publicaciones y cambios públicos desde rastros secundarios.

## Dinámica de resolución
El alumno correlaciona changelog, caché y nota editorial para reconstruir la publicación original.

## Material del alumno
- Bundle final: `huella_de_publicacion_bundle.zip`
- Página de apoyo: `/huella-de-publicacion`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{huella_de_publicacion_reconstruida}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto de lectura comparativa y precisión documental.
