# Datos para CTFd: Metadatos en cascada

        ## Name
        Metadatos en cascada

        ## Category
        OSINT

        ## Value
        620

        ## Type
        standard

        ## State
        visible

        ## Description final lista para pegar en CTFd
        Una serie de imágenes del laboratorio fue exportada varias veces y dejó una cadena de metadatos más útil de lo que parece. Debes identificar la herramienta de Kali adecuada, el editor responsable y la pista de ubicación que se repite entre archivos.

        Material de apoyo relacionado: [Guía interna: Metadatos en cascada](/metadatos-en-cascada).
        Descarga el material adjunto del reto y trabaja sobre el bundle sin modificar archivos del organizador.

        ## Flag
        `CUH{exiftool_enlaza_la_historia}`

        ## Hints
        - [20] Este reto no va de observar solo la imagen, sino lo que el archivo cuenta sobre sí mismo.
- [35] Busca campos que sobreviven entre exportaciones: autor, software, comentarios o ubicación interna.
- [50] La respuesta final pide herramienta, editor y pista de ubicación. La correlación sale de varios metadatos, no de uno solo.
