# Repositorio fantasma

## Propósito didáctico
Practicar atribución de repositorios retirados mediante rastros documentales parciales.

## Dinámica de resolución
El alumno cruza capturas y exportes para reconstruir repo, organización y versión.

## Material del alumno
- Bundle final: `repositorio_fantasma_bundle.zip`
- Página de apoyo: `/repositorio-fantasma`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{repositorio_fantasma_atribuido}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto OSINT de correlación multifuente, no de búsqueda directa.
