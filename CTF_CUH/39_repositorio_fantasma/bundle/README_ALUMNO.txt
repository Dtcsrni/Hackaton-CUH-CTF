Repositorio fantasma
==========================

Objetivo
--------
Atribuye correctamente el repositorio retirado, su organización y el tag asociado.

Qué contiene el bundle
----------------------
Captura antigua, export de issues, changelog y validador.

Qué debes entregar
------------------
Completa `respuesta.txt` y ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
