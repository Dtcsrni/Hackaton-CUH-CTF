# Datos para CTFd: Agenda filtrada

        ## Name
        Agenda filtrada

        ## Category
        OSINT

        ## Value
        390

        ## Type
        standard

        ## State
        visible

        ## Description final lista para pegar en CTFd
        El material incluye una agenda parcial, una nota de sala y un extracto de convocatoria. Debes reconstruir qué taller coincide con las tres piezas, en qué sala ocurrirá y a qué hora arranca.

        Material de apoyo relacionado: [Guía interna: Agenda filtrada](/agenda-filtrada).
        Descarga el material adjunto del reto y trabaja sobre el bundle sin modificar archivos del organizador.

        ## Flag
        `CUH{agenda_filtrada_reconstruida}`

        ## Hints
        - [20] La sala no aparece igual en todos los documentos; busca abreviaturas y equivalencias.
- [35] El taller correcto es el único que cuadra a la vez con tema, franja y nota de sala.
- [50] No entregues todo el horario: solo taller, sala y hora de inicio.
