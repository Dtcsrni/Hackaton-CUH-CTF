Contenido mixto heredado
==========================

Objetivo
--------
Corrige el contenido mixto para que el portal cargue recursos y API solo por HTTPS.

Qué contiene el bundle
----------------------
Plantilla HTML, frontend JS, nota de contexto y validador.

Qué debes entregar
------------------
Edita `web/dashboard.html` y `web/app.js`, luego ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
