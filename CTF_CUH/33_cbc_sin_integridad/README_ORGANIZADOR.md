# CBC sin integridad

## Propósito didáctico
Practicar la diferencia entre cifrado con confidencialidad y cifrado autenticado.

## Dinámica de resolución
El alumno revisa evidencias, identifica AES-CBC sin autenticación y completa la respuesta estructurada.

## Material del alumno
- Bundle final: `cbc_sin_integridad_bundle.zip`
- Página de apoyo: `/cbc-sin-integridad`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{cbc_sin_integridad_identificada}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto conceptual de criptografía aplicada y diseño seguro de mensajes.
