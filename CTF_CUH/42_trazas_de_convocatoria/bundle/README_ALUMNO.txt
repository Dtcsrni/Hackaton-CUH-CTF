Trazas de convocatoria
==========================

Objetivo
--------
Atribuye correctamente el documento maestro, el editor y la clave de lote.

Qué contiene el bundle
----------------------
Versión archivada, propiedades, nota de distribución, publicación secundaria y validador.

Qué debes entregar
------------------
Completa `respuesta.txt` y ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
