# Trazas de convocatoria

## Propósito didáctico
Practicar atribución documental avanzada a partir de metadatos, renombres y versiones archivadas.

## Dinámica de resolución
El alumno cruza propiedades, notas y versiones para reconstruir el documento maestro.

## Material del alumno
- Bundle final: `trazas_de_convocatoria_bundle.zip`
- Página de apoyo: `/trazas-de-convocatoria`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{convocatoria_atribuida_y_validada}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto OSINT de mayor dificultad, orientado a trazabilidad documental.
