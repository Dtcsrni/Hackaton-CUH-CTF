# Datos para CTFd: Binario de despacho

        ## Name
        Binario de despacho

        ## Category
        Reversing

        ## Value
        580

        ## Type
        standard

        ## State
        visible

        ## Description final lista para pegar en CTFd
        Se entrega un binario de escritorio usado en el laboratorio para validar un código interno. Debes analizarlo y recuperar la clave correcta que el binario espera.

        Material de apoyo relacionado: [Guía interna: Binario de despacho](/binario-de-despacho).
        Descarga el material adjunto del reto y trabaja sobre el bundle sin modificar archivos del organizador.

        ## Flag
        `CUH{clave_de_despacho_recuperada}`

        ## Hints
        - [20] Empieza por strings y por el flujo principal de validación.
- [35] No necesitas parchear el binario; solo entender qué formato espera.
- [50] La clave final se puede reconstruir observando comparaciones simples y el formato completo.
