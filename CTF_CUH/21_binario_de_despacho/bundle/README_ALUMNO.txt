Binario de despacho
==========================

Objetivo
--------
Analiza el ejecutable y recupera el código exacto que considera válido.

Qué contiene el bundle
----------------------
Binario PE, nota breve y validador de respuesta.

Qué debes entregar
------------------
Escribe el código en `respuesta.txt` y valida localmente.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
