# Perfil disperso

## Propósito didáctico
Practicar correlación básica de identidad y perfil profesional con artefactos sintéticos.

## Dinámica de resolución
El alumno revisa tres fuentes breves, correlaciona datos y entrega un archivo de respuestas.

## Material del alumno
- Bundle final: `perfil_disperso_bundle.zip`
- Página de apoyo: `/perfil-disperso`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{perfil_disperso_correlacionado}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Todo el contenido es ficticio y está pensado para lectura comparativa.
