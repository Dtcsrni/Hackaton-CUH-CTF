# Incidente en formularios

## Propósito didáctico
Practicar reconstrucción forense sobre un incidente de autenticación sin depender de explotación activa.

## Dinámica de resolución
El alumno revisa evidencias, llena un archivo de respuestas y valida localmente.

## Material del alumno
- Bundle final: `incidente_en_formularios_bundle.zip`
- Página de apoyo: `/incidente-en-formularios`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{incidente_de_formularios_reconstruido}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Es un reto de análisis puro; no requiere modificar código.
