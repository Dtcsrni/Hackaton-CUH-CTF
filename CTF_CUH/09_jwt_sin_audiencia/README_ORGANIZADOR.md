# JWT sin audiencia

## Propósito didáctico
Reforzar validación completa de JWT desde el backend.

## Dinámica de resolución
El alumno corrige un validador Python sencillo y ejecuta la prueba local.

## Material del alumno
- Bundle final: `jwt_sin_audiencia_bundle.zip`
- Página de apoyo: `/jwt-sin-audiencia`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{jwt_con_validacion_completa}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Es un reto de autenticación defensiva, no de creación de tokens maliciosos.
