# Datos para CTFd: Recuperación de contexto

        ## Name
        Recuperación de contexto

        ## Category
        IA defensiva

        ## Value
        520

        ## Type
        standard

        ## State
        visible

        ## Description final lista para pegar en CTFd
        Se entrega un lote de trazas de un asistente que terminó exponiendo contexto interno. Debes identificar la causa inmediata, el dato sensible filtrado y la mitigación de mayor impacto.

        Material de apoyo relacionado: [Guía interna: Recuperación de contexto](/recuperacion-de-contexto).
        Descarga el material adjunto del reto y trabaja sobre el bundle sin modificar archivos del organizador.

        ## Flag
        `CUH{contexto_filtrado_y_mitigado}`

        ## Hints
        - [20] Primero revisa el orden de ensamblado de mensajes en la traza.
- [35] No busques un exploit; busca la razón por la que el contexto terminó mezclado con la respuesta.
- [50] La respuesta final debe nombrar causa, dato filtrado y mitigación concreta.
