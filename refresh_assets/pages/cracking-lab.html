Hashes, ZIP protegidos y credenciales débiles del paquete CUH

Esta página acompaña retos concretos como Hash filtrado, ZIP bajo llave, Acceso heredado, Acceso por defecto y Formulario de acceso. Aquí el punto no es lanzar diccionarios gigantes, sino reconocer si tienes un PIN corto, una contraseña débil, un Basic Auth o un formulario POST con parámetros claros.

Tipos de material esperable

  • Hashes cortos o PIN simples.
  • ZIP con contraseña débil.
  • Diccionarios o pistas que acotan la búsqueda.

Criterio

Primero identifica formato y superficie: hash offline, ZIP cifrado, Basic Auth o login web. Solo después eliges la técnica. En este laboratorio la dificultad está acotada deliberadamente para que la lectura del material valga más que el volumen de ataque.
