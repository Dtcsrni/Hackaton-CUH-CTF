Inventario técnico

Servicios reales que forman parte del laboratorio CUH

Esta referencia cruza puertos y rutas con retos concretos. Si un enunciado menciona la IP 45.55.49.111 pero no sabes si debes usar nmap, curl o nc, aquí puedes ubicar qué tipo de servicio existe y qué reto lo usa.

TCP y HTTP de reconocimiento

  • 45.55.49.111:31337 -> Puertas abiertas, banner TCP accesible con nc.
  • http://45.55.49.111:8081 -> Cabeceras del laboratorio, la pista vive en headers HTTP.
  • http://45.55.49.111:8082 -> JSON de prueba, conviene revisar / y /api/status.
  • http://45.55.49.111:8083/panel -> Acceso heredado, servicio web con credenciales débiles dentro del alcance.

Qué debes concluir a partir de esto

  • Si el reto solo da una IP, primero enumera.
  • Si el reto ya da una URL exacta, no necesitas escanear a ciegas todo el host.
  • Los puertos altos del evento no son ruido: suelen existir porque el objetivo es aprender a enumerar antes de interpretar.
  • Este inventario no reemplaza el enunciado; solo evita que confundas un reto de red con uno de navegador o con un adjunto local.
