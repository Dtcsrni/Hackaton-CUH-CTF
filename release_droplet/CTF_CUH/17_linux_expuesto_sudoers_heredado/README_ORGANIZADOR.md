# Linux expuesto: sudoers heredado

## Propósito didáctico
Trabajar reducción de privilegios y revisión de `sudoers`.

## Dinámica de resolución
El alumno corrige el archivo de política y valida localmente.

## Material del alumno
- Bundle final: `linux_expuesto_sudoers_heredado_bundle.zip`
- Página de apoyo: `/linux-expuesto-sudoers-heredado`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{sudoers_heredado_corregido}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
El reto enseña hardening de configuración, no abuso de comandos privilegiados.
