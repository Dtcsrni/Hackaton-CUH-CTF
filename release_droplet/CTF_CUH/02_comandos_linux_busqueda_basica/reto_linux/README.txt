Laboratorio de archivos - CUH

Este material corresponde a un entorno académico controlado para práctica de observación y búsqueda básica en Linux.
Revise la estructura con calma, identifique qué directorios contienen evidencia útil y documente sus hallazgos antes de enviar una respuesta.
No todos los archivos aportan el mismo valor; parte del ejercicio consiste en discriminar información relevante de notas operativas rutinarias.
