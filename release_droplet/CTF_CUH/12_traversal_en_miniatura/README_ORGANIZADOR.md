# Traversal en miniatura

## Propósito didáctico
Practicar hardening defensivo de un lector de archivos pequeño.

## Dinámica de resolución
El alumno corrige el backend Python y valida localmente.

## Material del alumno
- Bundle final: `traversal_en_miniatura_bundle.zip`
- Página de apoyo: `/traversal-en-miniatura`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{rutas_normalizadas_y_resueltas}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
El reto enseña control de alcance, no enumeración de rutas reales.
