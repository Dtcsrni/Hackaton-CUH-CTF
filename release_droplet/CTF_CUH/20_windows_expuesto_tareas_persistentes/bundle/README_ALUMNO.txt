Windows expuesto: tareas persistentes
==========================

Objetivo
--------
Reconstruye binario, cuenta de ejecución y acción inmediata de contención.

Qué contiene el bundle
----------------------
XML de tarea, eventos resumidos y validador.

Qué debes entregar
------------------
Completa `respuesta.txt` y ejecútalo contra el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
