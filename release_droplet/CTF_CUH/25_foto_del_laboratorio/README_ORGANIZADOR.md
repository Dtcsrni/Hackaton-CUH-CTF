# Foto del laboratorio

## Propósito didáctico
Practicar correlación entre observación visual, metadatos y documentos de inventario.

## Dinámica de resolución
El alumno cruza tres evidencias y responde con identificador, ubicación y detalle confirmatorio.

## Material del alumno
- Bundle final: `foto_del_laboratorio_bundle.zip`
- Página de apoyo: `/foto-del-laboratorio`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{foto_del_laboratorio_interpretada}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
La evidencia es textual y sintética; no depende de imágenes pesadas ni herramientas externas.
