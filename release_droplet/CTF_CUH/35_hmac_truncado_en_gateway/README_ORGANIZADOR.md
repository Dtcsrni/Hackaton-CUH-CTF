# HMAC truncado en gateway

## Propósito didáctico
Practicar implementación correcta de HMAC y verificación de firmas hexadecimales.

## Dinámica de resolución
El alumno endurece la función verificadora y valida el resultado con pruebas locales.

## Material del alumno
- Bundle final: `hmac_truncado_en_gateway_bundle.zip`
- Página de apoyo: `/hmac-truncado-en-gateway`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{gateway_hmac_verificado_completo}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto de parche con foco en comparación completa y segura.
