# Reportes sin parámetros

## Propósito didáctico
Enseñar parametrización de búsquedas con LIKE sin caer en patrones inseguros.

## Dinámica de resolución
El alumno parchea el archivo PHP principal del buscador y valida la corrección localmente.

## Material del alumno
- Bundle final: `reportes_sin_parametros_bundle.zip`
- Página de apoyo: `/reportes-sin-parametros`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{reportes_filtrados_con_parametros}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
El material del alumno contiene solo el backend relevante y un esquema mínimo para dar contexto.
