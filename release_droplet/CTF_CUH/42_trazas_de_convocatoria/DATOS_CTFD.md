# Datos para CTFd: Trazas de convocatoria

        ## Name
        Trazas de convocatoria

        ## Category
        OSINT

        ## Value
        680

        ## Type
        standard

        ## State
        visible

        ## Description final lista para pegar en CTFd
        Una convocatoria aparentemente limpia fue republicada varias veces y dejó una huella dispersa entre PDFs, propiedades de documento y versiones archivadas. Debes atribuir la versión maestra, el editor responsable y la clave interna usada para nombrar el lote.

        Material de apoyo relacionado: [Guía interna: Trazas de convocatoria](/trazas-de-convocatoria).
        Descarga el material adjunto del reto y trabaja sobre el bundle sin modificar archivos del organizador.

        ## Flag
        `CUH{convocatoria_atribuida_y_validada}`

        ## Hints
        - [20] Las versiones no solo cambian en el texto visible; también cambian en propiedades y nombres internos.
- [35] Cruza PDF, versión archivada y nota de distribución. La atribución correcta sale de esa intersección.
- [50] Necesitas documento maestro, editor y clave de lote. Si uno depende solo de una fuente débil, vuelve a correlacionar.
