# Prompt de soporte indiscreto

## Propósito didáctico
Estudiar riesgos de prompt design desde un enfoque defensivo y verificable.

## Dinámica de resolución
El alumno endurece el ensamblado del prompt y valida la mitigación localmente.

## Material del alumno
- Bundle final: `prompt_de_soporte_indiscreto_bundle.zip`
- Página de apoyo: `/prompt-de-soporte-indiscreto`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{prompt_de_soporte_endurecido}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
No se incluye ningún proveedor externo; todo ocurre en el código del bundle.
