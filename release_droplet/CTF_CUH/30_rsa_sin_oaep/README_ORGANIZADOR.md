# RSA sin OAEP

## Propósito didáctico
Reforzar migración segura de RSA heredado hacia OAEP.

## Dinámica de resolución
El alumno corrige el módulo Python y valida la política resultante.

## Material del alumno
- Bundle final: `rsa_sin_oaep_bundle.zip`
- Página de apoyo: `/rsa-sin-oaep`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{rsa_con_oaep_y_sha256}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto de revisión y parche, no de explotación ni de criptografía ofensiva.
