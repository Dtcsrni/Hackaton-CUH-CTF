Red de proveedores
==========================

Objetivo
--------
Identifica la empresa pivote, el dominio compartido y el documento que conecta la red.

Qué contiene el bundle
----------------------
Invitación, firma, nota de relaciones, resumen público y validador.

Qué debes entregar
------------------
Completa `respuesta.txt` y ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
