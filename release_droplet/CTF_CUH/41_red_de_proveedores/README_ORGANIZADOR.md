# Red de proveedores

## Propósito didáctico
Practicar mapeo de redes de proveedores y atribución por dominio y documentación conectiva.

## Dinámica de resolución
El alumno detecta el nodo pivote y sintetiza la relación central.

## Material del alumno
- Bundle final: `red_de_proveedores_bundle.zip`
- Página de apoyo: `/red-de-proveedores`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{red_de_proveedores_mapeada}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto OSINT más estratégico, basado en correlación entre entidades y documentos.
