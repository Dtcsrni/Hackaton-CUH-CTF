Portal defaceado en PHP
==========================

Objetivo
--------
Reconstruye el archivo afectado, el stack del portal y la acción inmediata más razonable.

Qué contiene el bundle
----------------------
HTML comprometido, diff parcial, nota operativa y validador.

Qué debes entregar
------------------
Completa `respuesta.txt` y valida localmente.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
