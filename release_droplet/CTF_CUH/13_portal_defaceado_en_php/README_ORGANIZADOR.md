# Portal defaceado en PHP

## Propósito didáctico
Estudiar un deface como incidente forense y de respuesta, no como acción ofensiva.

## Dinámica de resolución
El alumno inspecciona evidencias y completa un archivo de respuestas.

## Material del alumno
- Bundle final: `portal_defaceado_en_php_bundle.zip`
- Página de apoyo: `/portal-defaceado-en-php`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{deface_reconstruido_y_contenido}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto pensado para análisis, documentación y criterio de contención.
