Carving de evidencias
==========================

Objetivo
--------
Identifica la herramienta adecuada, el archivo recuperado importante y el detalle operativo asociado.

Qué contiene el bundle
----------------------
Índice de carving, archivo recuperado, nota de recuperación y validador.

Qué debes entregar
------------------
Completa `respuesta.txt` y ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
