# Binario de despacho

## Propósito didáctico
Introducir reversing de binarios sencillos orientado a reconstrucción de lógica.

## Dinámica de resolución
El alumno analiza el ejecutable, recupera el código y valida la respuesta localmente.

## Material del alumno
- Bundle final: `binario_de_despacho_bundle.zip`
- Página de apoyo: `/binario-de-despacho`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{clave_de_despacho_recuperada}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
El binario se compila localmente durante la generación del reto y se distribuye como parte del bundle.
