# Agenda filtrada

## Propósito didáctico
Practicar correlación de documentos y extracción de datos relevantes desde texto operativo.

## Dinámica de resolución
El alumno revisa tres piezas, reconstruye el evento correcto y valida localmente.

## Material del alumno
- Bundle final: `agenda_filtrada_bundle.zip`
- Página de apoyo: `/agenda-filtrada`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{agenda_filtrada_reconstruida}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
El reto está pensado para lectura atenta y síntesis precisa.
