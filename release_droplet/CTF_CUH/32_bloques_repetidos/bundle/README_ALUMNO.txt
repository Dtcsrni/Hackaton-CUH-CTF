Bloques repetidos
==========================

Objetivo
--------
Identifica el modo de operación, el hallazgo observable y la mitigación principal.

Qué contiene el bundle
----------------------
Volcado hexadecimal, nota visual, contexto y validador.

Qué debes entregar
------------------
Completa `respuesta.txt` y ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
