# Bloques repetidos

## Propósito didáctico
Practicar identificación de modos inseguros a partir de patrones visibles en el cifrado.

## Dinámica de resolución
El alumno analiza el volcado hexadecimal, lee el contexto y sintetiza una respuesta precisa.

## Material del alumno
- Bundle final: `bloques_repetidos_bundle.zip`
- Página de apoyo: `/bloques-repetidos`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{bloques_repetidos_interpretados}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto de análisis de modo de operación; no requiere cálculo intensivo.
