# IV reciclado en reportes

## Propósito didáctico
Practicar análisis de IV reutilizado y consecuencias observables en CBC.

## Dinámica de resolución
El alumno compara artefactos, correlaciona prefijos y sintetiza una respuesta estructurada.

## Material del alumno
- Bundle final: `iv_reciclado_en_reportes_bundle.zip`
- Página de apoyo: `/iv-reciclado-en-reportes`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{iv_reciclado_detectado_en_reportes}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto de análisis orientado a patrones, no a descifrado completo.
