Restablecimiento abierto
==========================

Objetivo
--------
Endurece el flujo de restablecimiento para que el token sea aleatorio, temporal y vinculado al usuario correcto.

Qué contiene el bundle
----------------------
Código Python del flujo y contexto mínimo.

Qué debes entregar
------------------
Corrige `app/reset_flow.py` y ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
