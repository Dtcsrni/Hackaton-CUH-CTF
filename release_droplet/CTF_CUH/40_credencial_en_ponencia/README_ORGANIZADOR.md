# Credencial en ponencia

## Propósito didáctico
Practicar identificación de personas con fragmentos parciales y descarte de falsos positivos.

## Dinámica de resolución
El alumno combina varias piezas, elimina candidatos y responde con identidad y track.

## Material del alumno
- Bundle final: `credencial_en_ponencia_bundle.zip`
- Página de apoyo: `/credencial-en-ponencia`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{credencial_en_ponencia_correlacionada}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto OSINT de atribución con homónimos parciales y validación contextual.
