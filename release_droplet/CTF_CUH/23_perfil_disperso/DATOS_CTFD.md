# Datos para CTFd: Perfil disperso

        ## Name
        Perfil disperso

        ## Category
        OSINT

        ## Value
        360

        ## Type
        standard

        ## State
        visible

        ## Description final lista para pegar en CTFd
        Recibes varias capturas y textos breves de un perfil académico repartido entre distintas plataformas del evento. Debes correlacionarlos para identificar alias, correo institucional y la pista que conecta todo el conjunto.

        Material de apoyo relacionado: [Guía interna: Perfil disperso](/perfil-disperso).
        Descarga el material adjunto del reto y trabaja sobre el bundle sin modificar archivos del organizador.

        ## Flag
        `CUH{perfil_disperso_correlacionado}`

        ## Hints
        - [20] Empieza por lo repetido: alias, dominio de correo o misma frase en más de un artefacto.
- [35] No busques una sola fuente perfecta; el reto está en unir detalles parciales.
- [50] La respuesta final pide alias, correo y palabra de enlace, no una biografía completa.
