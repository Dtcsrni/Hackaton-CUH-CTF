Perfil disperso
==========================

Objetivo
--------
Relaciona los tres artefactos para identificar alias, correo institucional y palabra de enlace.

Qué contiene el bundle
----------------------
Tres perfiles sintéticos y un validador local.

Qué debes entregar
------------------
Completa `respuesta.txt` y ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
