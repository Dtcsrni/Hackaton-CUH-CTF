# Sesión que confía demasiado

## Propósito didáctico
Mostrar un error clásico de autorización por confiar demasiado en la sesión enviada por el cliente.

## Dinámica de resolución
El alumno corrige un archivo Python pequeño y valida la lógica.

## Material del alumno
- Bundle final: `sesion_que_confia_demasiado_bundle.zip`
- Página de apoyo: `/sesion-que-confia-demasiado`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{rol_de_sesion_validado_en_backend}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
El reto está pensado como revisión defensiva de backend, no como explotación.
