# Cronología cruzada

## Propósito didáctico
Practicar reconstrucción cronológica y correlación de múltiples huellas públicas.

## Dinámica de resolución
El alumno ordena evidencias, identifica el evento pivote y responde con una terna estructurada.

## Material del alumno
- Bundle final: `cronologia_cruzada_bundle.zip`
- Página de apoyo: `/cronologia-cruzada`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{cronologia_cruzada_reconstruida}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto OSINT de correlación temporal, más exigente que una búsqueda puntual.
