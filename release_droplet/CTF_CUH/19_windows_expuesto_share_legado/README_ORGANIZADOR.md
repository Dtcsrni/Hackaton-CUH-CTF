# Windows expuesto: share legado

## Propósito didáctico
Practicar lectura de ACLs y reducción de exposición en comparticiones Windows.

## Dinámica de resolución
El alumno revisa artefactos exportados y completa un archivo de respuestas.

## Material del alumno
- Bundle final: `windows_expuesto_share_legado_bundle.zip`
- Página de apoyo: `/windows-expuesto-share-legado`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{share_legado_reducido}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
No requiere laboratorio Windows vivo; todo es análisis de evidencia.
