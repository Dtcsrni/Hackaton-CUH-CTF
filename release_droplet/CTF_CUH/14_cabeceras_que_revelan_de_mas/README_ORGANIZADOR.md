# Cabeceras que revelan de más

## Propósito didáctico
Practicar revisión de configuración de borde y cabeceras de seguridad.

## Dinámica de resolución
El alumno corrige el bloque de nginx y valida el resultado.

## Material del alumno
- Bundle final: `cabeceras_que_revelan_de_mas_bundle.zip`
- Página de apoyo: `/cabeceras-que-revelan-de-mas`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{cabeceras_endurecidas}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto estático de hardening; no depende de un nginx vivo.
