# Diccionario de laboratorio

## Propósito didáctico
Practicar uso razonado de herramientas de cracking de Kali sin convertir el reto en una carrera ciega de fuerza bruta.

## Dinámica de resolución
El alumno lee política, prioriza cuenta y wordlist y responde con la estrategia inicial correcta.

## Material del alumno
- Bundle final: `diccionario_de_laboratorio_bundle.zip`
- Página de apoyo: `/diccionario-de-laboratorio`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{john_prioriza_el_contexto}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto seguro de cracking controlado y selección de herramientas/contexto.
