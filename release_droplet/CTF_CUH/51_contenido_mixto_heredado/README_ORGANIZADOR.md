# Contenido mixto heredado

## Propósito didáctico
Practicar limpieza de contenido mixto después de migraciones parciales a HTTPS.

## Dinámica de resolución
El alumno revisa HTML y JS, actualiza URLs heredadas y valida que desaparezcan todas las referencias inseguras.

## Material del alumno
- Bundle final: `contenido_mixto_heredado_bundle.zip`
- Página de apoyo: `/contenido-mixto-heredado`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{todos_los_recursos_van_por_https}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto de remediación simple pero muy realista en despliegues web legacy.
