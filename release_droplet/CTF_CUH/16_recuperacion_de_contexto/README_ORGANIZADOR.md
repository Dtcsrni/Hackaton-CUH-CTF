# Recuperación de contexto

## Propósito didáctico
Practicar análisis forense de fugas de contexto en sistemas asistidos por modelos.

## Dinámica de resolución
El alumno lee trazas, completa un archivo de respuestas y valida localmente.

## Material del alumno
- Bundle final: `recuperacion_de_contexto_bundle.zip`
- Página de apoyo: `/recuperacion-de-contexto`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{contexto_filtrado_y_mitigado}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto de análisis defensivo; no requiere proveedor ni modelo real.
