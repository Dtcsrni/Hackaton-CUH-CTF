Subida de archivos ansiosa
==========================

Objetivo
--------
Haz que la subida valide extensión y tamaño, use nombre generado por el servidor y guarde en un directorio privado.

Qué contiene el bundle
----------------------
Handler Python de subida y requisitos mínimos.

Qué debes entregar
------------------
Edita `app/upload_handler.py` y valida localmente.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
