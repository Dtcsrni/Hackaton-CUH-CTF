# Metadatos en cascada

## Propósito didáctico
Practicar extracción y correlación de metadatos con herramientas habituales de Kali como exiftool.

## Dinámica de resolución
El alumno identifica autoría y ubicación a partir de varios campos persistentes entre exportaciones.

## Material del alumno
- Bundle final: `metadatos_en_cascada_bundle.zip`
- Página de apoyo: `/metadatos-en-cascada`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{exiftool_enlaza_la_historia}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto OSINT técnico, más centrado en archivos que en navegación pública.
