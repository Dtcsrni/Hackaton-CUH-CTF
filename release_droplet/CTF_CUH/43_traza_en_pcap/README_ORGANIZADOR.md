# Traza en PCAP

## Propósito didáctico
Practicar análisis rápido de capturas usando herramientas típicas de Kali como tshark.

## Dinámica de resolución
El alumno revisa el resumen de tráfico, identifica el host y el recurso útil y responde con una terna estructurada.

## Material del alumno
- Bundle final: `traza_en_pcap_bundle.zip`
- Página de apoyo: `/traza-en-pcap`

## Validación del organizador
1. Ejecuta `python verify_organizer.py` dentro de esta carpeta.
2. Confirma que el validador devuelve `CUH{tshark_reconstruye_la_pista}`.
3. Sube el ZIP a CTFd como archivo descargable del reto.
4. Verifica que la página de apoyo publique el contexto correcto.

## Nota pedagógica
Reto orientado a lectura de red y priorización de artefactos, no a explotación.
