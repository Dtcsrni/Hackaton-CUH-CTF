Traza en PCAP
==========================

Objetivo
--------
Identifica la herramienta de Kali adecuada, el host relevante y el recurso descargado.

Qué contiene el bundle
----------------------
Resumen de captura, nota del SOC, contexto y validador.

Qué debes entregar
------------------
Completa `respuesta.txt` y ejecuta el validador.

Cómo validarlo
--------------
Ejecuta:

```powershell
python tests\validate_fix.py
```

Si la corrección o el análisis es consistente, el validador imprimirá la flag.
